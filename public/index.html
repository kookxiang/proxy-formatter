指令速查

查看完整说明
fetch / fetch-once 拉取订阅
include / exclude 过滤节点
replace / prefix / suffix 改名
header / proxy 设置请求行为
resolve / check-traffic 处理地址和流量
clash / surge [version] 输出格式

语法规则

  • 一行一个指令，按顺序执行。
  • 空行会忽略，行首 # 代表注释。
  • 参数支持双引号，适合带空格的值。
  • /.../ 会按正则处理，其余按普通字符串匹配。

结果面板
